# Contact-Form
Building a Contact Form using Angular Reactive Forms

- The form contains
  - First Name
  - Last Name
  - Email
  - Phone Number
  - Subject
  - Message
- Validation
  - All fields are mandatory
  - First Name & Last Name: Minimum 3 characters
  - Email: Valid email format
  - Phone Number: Exactly 10 digits
  - Subject: Minimum 5 characters
  - Message: Minimum 20 characters
 
- Submission
  - Validate all fields before submission
  - If the form is valid, display the entered information in a dialog/modal popup
  - If the form is invalid, show appropriate validation messages
 
- Additional Requirements
  - Add a Reset button
  - Create a clean and responsive UI
  - Disable the Submit button until the form becomes valid
  - Add a character counter for the Message field
  - Store submitted records and display them in a table
